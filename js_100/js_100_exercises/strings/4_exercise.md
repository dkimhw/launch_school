

### Multiline String

Question:

Take the following rhyme:

A pirate I was meant to be!
Trim the sails and roam the sea!

How can you assign this string to a single variable, preserving the line break?

Answer:

```javascript
let str = `
A pirate I was meant to be!
Trim the sails and roam the sea!
`

console.log(str);
```
